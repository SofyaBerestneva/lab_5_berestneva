# lab_5_berestneva

## Краткое описание

Код, который из данной последовательности чисел выводит список четных чисел больше 10 и список оставшихся чисел.

## Как использовать

1. Клонирование репозитория 

```git clone https://github.com/SofyaBerestneva/lab_5_berestneva.git```

2. Перейдите в директорию project (если Вы не в ней)

```cd project```

3. Выполнение

```python file1.py```
 
 ## Зависимости
Эта программа зависит от интепретатора Python версии 3.11 или выше.
